



🚗 **AI-Based Vehicle Number Plate Detection & Owner Identification System**

I recently built a computer vision project that automates **vehicle number plate detection and information retrieval** using deep learning and OCR.

🔍 **What the system does:**

* Detects vehicle number plates using a custom-trained **YOLO model (best.pt)**
* Extracts text from the plate using **EasyOCR**
* Cleans and processes the detected number using regex
* Fetches vehicle owner details from a **MySQL database**
* Displays results in real-time with bounding box visualization

⚙️ **Tech Stack Used:**

* Python 🐍
* YOLO (Ultralytics) for object detection
* OpenCV for image processing
* EasyOCR for text recognition
* MySQL for database integration

💡 **Key Features:**
✔ High-speed number plate detection
✔ Automated data retrieval system
✔ Image preprocessing for improved OCR accuracy
✔ End-to-end pipeline from detection → recognition → database lookup

📌 **Use Cases:**

* Smart parking systems
* Traffic monitoring
* Automated toll collection
* Security & surveillance

This project helped me gain hands-on experience in **deep learning, computer vision, and database integration**.

Looking forward to improving this further with **real-time video processing and higher OCR accuracy** 🚀
